
🏠 Locumie – Aplikacja Społecznościowa
Locumie to nowoczesna aplikacja mobilna zbudowana we Flutterze, która ułatwia nawiązywanie kontaktów między osobami poszukującymi zakwaterowania a wynajmującymi w najbliższej okolicy. Dzięki interaktywnej mapie, użytkownicy mogą błyskawicznie lokalizować oferty i komunikować się bezpośrednio przez aplikację.

🚀 Kluczowe Funkcje
Interaktywna Mapa: Lokalizowanie osób poprzez geolokalizacje, gdy osoba znajduje się niedaleko można napisać do niej

System Wiadomości: Możliwość wysyłania bezpośrednich komunikatów po naciśnięciu ikony profilu na mapie.

Zarządzanie Stanem: Zaawansowana obsługa stanów aplikacji zapewniająca płynność działania interfejsu.

Integracja API: Komunikacja z zewnętrznymi usługami i bazami danych dla zapewnienia aktualności ofert.

🛠️ Stack Technologiczny
Frontend: Flutter & Dart.

Backend/Baza Danych: Firebase (Realtime Database / Firestore) oraz Kotlin.

Geolokalizacja: Niestandardowa mapa bez użycia Google Maps

Pobierz zależności: flutter pub get

Uruchom aplikację: flutter run
